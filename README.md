# go2048
golang 实现console版2048游戏   

## 安装使用
```
//下载并安装依赖
$go get -u github.com/cwen-coder/go2048 

//运行程序
$cd $GOPATH/src/github.com/cwen-coder/go2048 
$go run go2048.go
```
